Privacy Policy

Last updated: February 8, 2026

1. Information We Collect

We collect only the information required to operate the Service.

OAuth & Token Data

This includes access tokens, refresh tokens, token expiration data, and associated service identifiers obtained through OAuth authorization.

API Usage Data

We collect API key identifiers, request timestamps, and usage metadata for security and rate limiting.

Contact Information

If provided, we may store your email address for access keys, support, or service-related communication.

2. How We Use Data

We use collected data to authenticate requests, refresh tokens, serve tokens securely to authorized clients, prevent abuse, and operate the Service.

We do not sell user data.

3. Token Storage & Security

Tokens are stored securely using encryption and access controls appropriate to their sensitivity.

Refresh tokens are used only to obtain new access tokens and are never shared with unauthorized parties.

4. Data Sharing

We do not share data except with authorized services, when required by law, or to protect the security and integrity of the Service.

5. Data Retention

Token data is retained only as long as necessary to provide the Service. Tokens may be deleted after access is revoked.

6. Your Choices

You may revoke OAuth access through the relevant third-party provider or stop using the Service at any time.

7. Changes

We may update this Privacy Policy periodically. Changes will be reflected by the last updated date.

8. Contact

For privacy-related questions, contact: support@curlmate.dev
